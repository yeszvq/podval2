Как заполняются файлы для диалогов?

Структура строки в которой залюкчен только текст:
Название текстуры для изображенияя#Имя персонажа#Текст

Структура строки выбора:
chose#[Имя выбора]||[Имя эффекта]||[значение]#[Имя выбора 2]||[Имя эффекта]||[значение]#....

Такая строка выведет панель выбора на которой будут выведены кнопки.
- Количество кнопок может быть любым
- У любой кнопки может быть любой эффект и даже несколько таких эффектов

Перечень эффектов:
	addItem_[наименование предмета]_[количество] - добавляет в инвентарь указанное количество предметов. Указывать количество ОБЯЗАТЕЛЬНО, даже если количество будет 1!!!

	nothing - ничего не делает

	remove_[наименование предмета]_[количество]

	cutscene_[наименование катсцены] - запускает катсцену

	close - заканчивает диалог после этой строки

	cutsceneCont - продолжит воспроизведение анимации



Также к строке можно добавить пост-эффект, который будет выполнен сразу при запуске строки:
&[Название эффекта]_[значение]&[Название эффекта]_[значение]&.....

-Количество пост-эффектов может быть любым

Пост_эффекты такие же как и обычные эффекты
